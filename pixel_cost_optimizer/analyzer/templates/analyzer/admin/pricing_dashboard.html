📋 How Pricing Works:

  • Your custom rules take priority over system defaults
  • System defaults are configured by the admin for fallback pricing
  • Coverage percentage is calculated based on non-white pixels in the document
  • No overlapping ranges are allowed for the same print type

Common Coverage Ranges: Logo/Small Graphics (10-15%), Medium Content (20-30%), Half Page (50%), Heavy Coverage (60-70%), Full Page (100%)
