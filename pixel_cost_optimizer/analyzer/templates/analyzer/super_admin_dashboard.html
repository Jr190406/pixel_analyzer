Account Management Guidelines

  • Reset Password: Generates a new temporary password and sends it to the user's email
  • Deactivate Account: Disables the account without deleting data - user cannot log in
  • Activate Account: Re-enables a previously deactivated account
  • Delete Account: Permanently removes the user account and all associated data
  • Super Admin accounts cannot be deleted for security reasons
